Bridgeport CNS milling machine

Bidgeport upgraded to LinuxCNC

This is a Bridgeport CNC milling machine, circa 1987, that I aquired back in 2000. It served me very well until a problem in the controler arose, which I couldn't find the root cause. I then decided to upgrade it to Linux CNC.

Upgrade

This is a short list of the modifications

  • Removed the console
  • Disconnected all circuits related to the console
  • Change the controler for a PC
  • Added servo card and configule the Hall file for the LinuxCNC

Timeline

  1. Strip original controler: 2 days,
  2. Install Pentium computer in the console: 2 days
  3. Add screen and control buttons: 3 days
  4. Wire the new control: 4 days
